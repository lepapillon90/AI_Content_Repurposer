# 📈 02_PRODUCT_GROWTH_ROADMAP.md (Phase 2 & 3)

> **비전**: 단순한 도구에서 크리에이터와 마케터를 위한 포괄적인 "콘텐츠 운영 체제"로 진화합니다.

---

## 🗓️ Phase 2: 실제 AI 및 맞춤화 (1-2개월 차)

### 🔌 실제 AI 통합
- **목표**: 시뮬레이션을 실제 LLM 기능으로 대체합니다.
- **작업**:
    - OpenAI API (GPT-4o) 또는 Gemini Pro API 통합.
    - 실시간 타이핑 효과를 위한 스트림 응답 구현.
    - "어조(Tone of Voice)" 선택기 추가 (전문적인, 재치 있는, 캐주얼한, 긴급한).

### 👤 사용자 경험 업그레이드
- **히스토리 탭**: 최근 생성 내용을 로컬에 저장 (LocalStorage).
- **편집 모드**: 복사하기 전에 생성된 결과를 수동으로 수정할 수 기능.
- **원클릭 공유**: Twitter/LinkedIn 작성기를 직접 여는 딥링크.

---

## 🗓️ Phase 3: 계정 및 수익화 (3-6개월 차)

### 🔐 사용자 시스템
- **인증**: Firebase Authentication (Google/이메일).
- **클라우드 동기화**: 기기 간 콘텐츠 기록 저장 (Firestore).

### 💎 수익화 (부분 유료화 모델)
- **무료 등급**: 일일 5회 생성, 표준 모델.
- **프로 등급 ($9/월)**:
    - 무제한 생성.
    - 고급 모델 (GPT-4).
    - "브랜드 보이스" 맞춤 지침.
    - 대량 변환 (원클릭 -> 모든 플랫폼).

---

## 🚀 마케팅 및 성장 전략
- **SEO**: "블로그를 트위터로 변환", "링크드인 포스트 생성기" 등을 위한 랜딩 페이지 생성.
- **바이럴 메카니즘**: 무료 등급 결과물에 "Made with RepurposeAI" 워터마크 표시.
- **커뮤니티**: 프롬프트와 팁을 공유하는 콘텐츠 크리에이터 디스코드 개설.
