## ✅ 성공 기준 (MVP)
1.  **사용성**: 사용자가 텍스트를 붙여넣고, 플랫폼을 선택하고, 3초 이내에 결과를 얻을 수 있음.
2.  **비주얼**: 앱이 전문적이고 신뢰할 수 있어 보임 (일반적인 템플릿 같지 않음).
3.  **반응형**: 모바일 기기에서 완벽하게 작동함.
