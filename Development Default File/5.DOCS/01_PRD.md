# 📄 01_PRD.md (제품 요구사항 정의서)

> **프로젝트**: RepurposeAI
> **버전**: 1.0 (MVP)
> **상태**: 승인됨

---

## 1. 프로젝트 개요
**RepurposeAI**는 콘텐츠 크리에이터, 마케터, 창업가가 긴 형식의 콘텐츠 가치를 극대화할 수 있도록 돕는 웹 기반 도구입니다. 사용자는 블로그 게시물이나 기사를 붙여넣기만 하면 Twitter, LinkedIn, Instagram, YouTube에 최적화된 콘텐츠를 즉시 생성할 수 있습니다.

### 핵심 문제
크로스 플랫폼 콘텐츠 제작은 시간이 많이 소요됩니다. 크리에이터들은 하나의 글을 각 플랫폼의 고유한 스타일과 제약 조건에 맞춰 여러 형식으로 변환하는 데 어려움을 겪습니다.

### 솔루션
각 플랫폼의 뉘앙스를 이해하고 그에 따라 텍스트를 변환하는 AI 기반 "리퍼포징 엔진"입니다.

---

## 2. 타겟 사용자 (페르소나)
*   **1인 창업가**: 퍼스널 브랜딩을 구축하고 싶지만 소셜 미디어에 할애할 시간이 제한적인 사람.
*   **콘텐츠 마케터**: 회사의 블로그 게시물을 모든 채널에 효율적으로 배포해야 하는 사람.
*   **뉴스레터 작가**: 주간 이슈를 매일의 트윗으로 변환하고 싶은 사람.

---

## 3. 핵심 기능 (MVP)

### P0: 필수 (Must Have)
1.  **멀티 플랫폼 지원**:
    - Twitter 쓰레드 (분할 로직 + 훅)
    - LinkedIn 포스트 (전문적인 포맷팅)
    - Instagram 캡션 (해시태그 + 공백)
    - YouTube Short 대본 (시각적 신호 + 내레이션)
2.  **스마트 입력**: 마크다운 또는 일반 텍스트를 지원하는 대형 텍스트 영역.
3.  **원클릭 복사**: 포맷된 공백과 함께 클립보드에 콘텐츠 복사.
4.  **시뮬레이션 모드**: 즉각적인 데모를 위한 고품질 모의 응답 (API 키 보류 상태).

### P1: 중요 (Next Step)
*   실제 AI 통합 (OpenAI/Gemini).
*   어조(Tone of Voice) 선택.

---

## 4. 사용자 수락 기준 (UAC)
*   사용자는 UI 지연 없이 500자 이상의 글을 붙여넣을 수 있다.
*   "Twitter"를 선택하면 번호가 매겨진 쓰레드 형식(예: "1/5")이 생성된다.
*   "LinkedIn"을 선택하면 "핵심 요약" 섹션이 있는 구조화된 게시물이 생성된다.
*   "Generate" 버튼은 AI가 "생각하는" 것을 모방하기 위해 최소 1.5초 동안 로딩 상태를 표시한다.

---

## 5. 성공 지표
*   **가치 도달 시간**: 사용자가 사이트에 접속한 후 10초 이내에 첫 번째 결과를 얻음.
*   **복사율**: 생성된 결과가 클립보드로 복사되는 비율.
